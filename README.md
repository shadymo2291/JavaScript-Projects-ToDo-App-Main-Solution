## 🧠 Overview

### 🎯 The challenge

Build a fully interactive ToDo app where users can:

✔️ View optimal layout across all devices  
✔️ Experience hover states for interactive elements  
✔️ Add, complete, and delete todos  
✔️ Filter todos (**All / Active / Completed**)  
✔️ Clear completed tasks  
✔️ Toggle 🌞 Light / 🌙 Dark mode  
✔️ Persist data using **localStorage**  
✔️ Track remaining active todos
